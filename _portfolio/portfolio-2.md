excerpt: "A complete production‑grade RAG (Retrieval‑Augmented Generation) system that transforms the French CIQUAL nutritional database into an intelligent, multimodal food search engine.<br/> The project implements an end‑to‑end ETL pipeline that ingests, normalises, and embeds over 3,400 foods from ANSES’s official French Food Composition Table. Using PostgreSQL with the pgvector extension, it enables fast cosine similarity search across 384‑dimensional text embeddings. A modular RAG architecture—comprising a retriever, a prompt builder, and a local Ollama LLM client, provides both text‑only and multimodal (text + image) query endpoints via FastAPI. The system retrieves relevant food records, builds a context‑aware prompt, and generates accurate, evidence‑based nutritional answers, with support for vision‑language models.<br/>GitHub repo: [food-nutrition-semantic-search-rag-system](https://github.com/tantikristanti/Generative-AI-LLMs/tree/main/food-nutrition-semantic-search-rag)<br/><img src='/images/portfolio/food-nutrition-rag-multimodal.png'><br/>"
